An IoT-based system for automating student entry, book authentication, and energy-efficient lighting in libraries.

## Features
- **RFID-based book and student authentication**
- **IR sensors for entry/exit detection**
- **Real-time count display on LCD**
- **LDR-controlled lights for energy saving**
- **Buzzer alert for unauthorized access**

## Hardware Used
- Arduino Uno
- RFID Module (MFRC522)
- IR Sensors
- LDR (Light Dependent Resistor)
- LCD I2C Display (16x2)
- Buzzer
